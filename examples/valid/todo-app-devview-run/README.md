# Todo App DevView Golden Run

This fixture is a closed DevView example for `npm run test:examples`.

It includes the canonical `.devview` layout:

- `.devview/tree/` for Product, Project, Work, and Test trees
- `.devview/execution/` for Cycle Tree artifacts
- `.devview/control/` for Decision, Change, Impact, and Acceptance trees
- `.devview/evidence/` for Evidence Tree artifacts and attached evidence files
- `.devview/blueprint/` for backward-compatible RPD, WPD, VD, and ACEP state artifacts

The example intentionally stays tiny: a todo item can be added, persisted, displayed, verified, evidenced, and accepted by a user.
