# Native DevView Examples

Native examples start from a graph-source instead of recovering intent from an
existing legacy project.

Current example:

- `graph-notes-demo/`
  - tiny local project under `work/native/graph-notes-demo`
  - proves the same graph operation chain used by retrofit:
    `graph-source -> instruction pack -> local change -> graph delta`
- `graph-counter-demo/`
  - tiny local project under `work/native/graph-counter-demo`
  - proves the same graph operation chain on an actual code behavior change
