# Internal Legacy Boundary

This directory contains historical migration fixtures and archived example
material used by DevView compatibility tests, audit reports, and future cleanup
planning.

Material here is not part of the public examples surface. Keep references to it
inside tests, scripts, generated provenance, or migration reports that explicitly
need historical inputs.

Do not treat files in this directory as public DevView examples, active source
authority, approval, accepted evidence, runtime satisfaction, equivalence proof,
scope enforcement, CI enforcement, or graph update permission.
