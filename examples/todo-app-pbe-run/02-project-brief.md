# 02 Project Brief

## Product Goal

Create a small Todo app that supports basic task management and keeps data after refresh.

## Current Slice

Selected:

- add todo
- complete todo
- delete todo
- local persistence after refresh

Deferred:

- user accounts
- backend sync
- drag and drop ordering
- categories

## Acceptance Notes

The user must be able to review the app manually and confirm whether the behavior is acceptable. Codex may submit results for review, but cannot mark the product accepted.
